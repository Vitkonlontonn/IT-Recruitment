1. Chức năng chính
- User: 
    - Đăng nhập, đăng ký
    - Xem bài đăng tuyển dụng
    - Lọc bài đăng tuyển dụng
    - Apply vào bài đăng 
- Admin:
    - Quản lý bài đăng (Thêm, sửa, xóa)
    - Duyệt đơn ứng tuyển của user
    - CRM
